## Slide 6: Detection Metrics
**Quantitative Evaluation**

**Metrics:**
- **Precision:** True positives / (TP + FP)
- **Recall:** True positives / (TP + FN)
- **F1-Score:** 2 * (P * R) / (P + R)
- **Detection %:** Detected / Total

**Usage:**
- Performance evaluation
- Plugin comparison
- Framework validation
- Research metrics
